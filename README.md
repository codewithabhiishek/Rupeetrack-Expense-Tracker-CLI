# RUPEETRACK 💸

A premium, interactive CLI Python application to help students track their expenses, built with the Python standard library.

**Key Features:**
- **Local Persistence & Export:** Automatically saves data to JSON and exports to CSV files.
- **Budgeting & Visuals:** Set a monthly budget with visual progress bars and dynamic ANSI color-coding.
- **Spending Insights:** Tracks daily averages, top categories, monthly breakdowns, and calculates a rule-based Financial Health Score with smart advice.
- **Reports:** Generates formatted text reports (`expense_report_YYYY-MM-DD.txt`) locally.
- **CLI Utility:** Filter, delete, and view expenses easily via interactive selections.

**How to run:** Run `python3 expense_tracker.py` in your terminal.

*Future Improvement:* If I had more time, I would build a web frontend using Flask/FastAPI or design interactive visualizations using `matplotlib` to graph monthly trends.
